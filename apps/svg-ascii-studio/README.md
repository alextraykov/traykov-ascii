# SVG ASCII Studio

A standalone Astro app for turning an uploaded SVG into an extruded Three.js object, rendering it through an ASCII shader, tuning the turntable, and exporting the resulting object as code.

## Local commands

```sh
npm install
npm run dev
npm run build
```

The app is static and Vercel-ready. Set the Vercel project root to `apps/svg-ascii-studio`.

The same workbench is mounted inside the portfolio at `/svg-ascii-studio/` via `src/pages/svg-ascii-studio.astro`.
