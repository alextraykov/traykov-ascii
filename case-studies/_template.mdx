## Constraints

The rules every option had to satisfy:

- **Design tokens only** — no hardcoded values. See `src/tokens/tokens.css`.
- **Dark mode parity** — `.dark` class on `<html>` auto-updates all tokens.
- **Motion tokens** — durations and easings from `--motion-duration-*` / `--motion-ease-*`. Every enter has an exit. Respect `prefers-reduced-motion`.
- **No Tailwind** — pure CSS with BEM naming.
- **React 19 + Framer Motion + pure CSS** — the stack is fixed.
- **Content guidelines** — see `docs/content-guidelines/principles.md`.

Feature-specific constraints go here too.
